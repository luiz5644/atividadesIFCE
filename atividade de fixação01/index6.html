PARADIGMAS DE PROGRAMAÇÃO

Os paradigmas de programação são estilos ou abordagens diferentes para escrever programas de
computador. Cada paradigma tem suas próprias regras e diretrizes que ditam como um programa
deve ser estruturado. As linguagens de programação são projetadas para suportar um ou mais
paradigmas, e a escolha de uma linguagem muitas vezes determina o paradigma predominante no
desenvolvimento do software. Aqui estão alguns dos principais paradigmas de programação e as
linguagens associadas a eles:

  1. Programação Imperativa:

    • Linguagens Principais: C, C++, Java, Python (para alguns casos)
    • Características: Nesse paradigma, os programas são estruturados em torno de comandos
      que alteram o estado do programa. O foco está em "como" fazer algo.

  2. Programação Orientada a Objetos (POO):

    • Linguagens Principais: Java, C#, Python, Ruby
    • Características: POO se concentra na modelagem do mundo real em termos de objetos,
      que contêm dados e métodos para operar nesses dados. É baseado em quatro princípios:
      encapsulamento, herança, polimorfismo e abstração.

  3. Programação Funcional:

    • Linguagens Principais: Haskell, Lisp, Scala, JavaScript (para alguns casos).
    • Características: Programação funcional trata as funções como cidadãs de primeira classe e
      enfatiza a imutabilidade de dados. A computação é vista como a avaliação de funções
      matemáticas

  4. Programação Lógica:

    • Linguagens Principais: Prolog.
    • Características: Nesse paradigma, você descreve o que deseja alcançar, e o sistema de
      lógica resolve como fazê-lo. Ele é frequentemente usado em sistemas de IA e resolução de
      problemas complexos.

  5. Programação Baseada em Eventos:

    • Linguagens Principais: JavaScript, C# (para interfaces gráficas).
    • Características: Programação baseada em eventos se concentra na reação a eventos
      externos, como cliques do mouse ou entradas do teclado. É comum em desenvolvimento
      de aplicativos com interfaces gráficas.

  6. Programação Concorrente e Paralela:

    • Linguagens Principais: Go, Erlang, Java (para concorrência), CUDA (para paralelismo de
      GPU).
    • Características: Nesses paradigmas, programas são projetados para executar tarefas
      simultaneamente. A programação concorrente lida com múltiplas tarefas cooperando,
      a programação paralela executa tarefas simultaneamente para melhorar o
      desempenho.
